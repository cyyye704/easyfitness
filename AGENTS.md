# AGENTS.md

## 项目说明
这是一个名为 EasyFitness 的网页端小程序，使用 Vite + React + TypeScript 构建。

## 项目目标
构建一个饮食与训练记录工具。

核心功能：
- 记录每日体重、饮食、训练和睡眠。
- 饮食记录李勇自然语言，包括食物名称、热量和蛋白质。
- 自动汇总每日总热量和总蛋白质。
- 根据当天记录给出评价：加分 / 持平 / 扣分。
- 当前阶段数据只保存在 localStorage。
- 暂时不做后端、不做登录、不做数据库。

## 常用命令
- 启动开发服务器：npm run dev
- 构建项目：npm run build
- 代码检查：npm run lint

## 工程规则
- 使用 TypeScript。
- 组件保持清晰、可读，不要写得过大。
- 优先使用简单的 React hooks 管理状态。
- 不要擅自添加后端、数据库、用户登录、鉴权或付费服务。
- 不要添加不必要的第三方依赖。
- 页面优先适配手机端。
- UI 风格保持简洁、理性、克制、有工程感。
- 代码要保证可维护性，不要为了快速实现而写成一坨。

## 工作流程
- 在进行较大规模代码修改前，先说明实现计划。
- 修改完成后，总结改了哪些文件。
- 每次实现主要功能后，运行 npm run build。
- 如果构建报错，先修复所有构建错误。
- 不要在没有说明的情况下删除已有功能。
- 不要覆盖用户手写内容，除非明确说明原因。

## 当前 MVP 范围
第一版只需要完成：
1. 今日记录表单。
2. 食物记录新增和删除。
3. 每日热量、蛋白质自动汇总。
4. 训练和睡眠简单记录。
5. 历史记录列表。
6. localStorage 本地保存。
7. 简洁移动端页面。
8. 减脂通识知识模块

## 暂时不要做
- 不做账号登录。
- 不做云端同步。
- 不做数据库。
- 不做复杂图表。
- 不做社交功能。
- 不做 AI 自动分析。
- 不做多用户系统。
